Magento 2 extension to enable a four-level address selection: Country → Region → District → City.
Automatically updates dropdowns dynamically for accurate and structured address input.

District and City data are loaded from CSV files included within the module.
This is a test/demo module intended for development and learning purposes only, not for production use.
